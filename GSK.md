'What is the difference between a data engineer, data analyst and data scientist?
Although they're closely related, the roles of data engineer, data analyst and data scientist have distinctly different responsibilities.

Data engineers are builders. They create systems and tools that collect, store, and process large amounts of data. Their role is to set up the infrastructure that allows data scientists and data analysts to do their jobs.

In contrast, data scientists are investigators. They look for patterns, trends, and insights in large amounts of data. Their role is to understand how to use that data to reveal useful information.

Data analysts can be thought of as storytellers. They take the data and turn it into useful information that can help businesses make decisions. They visualise the data, making it easy to understand through charts and graphs.'

'How is data engineering used at GSK?
Data engineering has many different applications - if there's data associated with a task, a data engineer will be involved somewhere!

Data engineering provides data aggregation and advanced programming for the site's data extraction and analysis. It builds data aggregation engines that provide pipelines between data sources and programming packages to support data analysis, such as:

Collecting data - data engineers set up systems to collect all the information from these sources.
Storing data - they would create secure, organised storage locations for this data.
Cleaning data - they make sure the data is accurate and free from mistakes before scientists analyse it.
Drug safety reporting - also known as pharmacovigilance - is when patients report bad reactions to medicines. Data scientists & engineers are essential here because they help with:

Reporting issues - data engineers set up systems to collect these reports from doctors and patients.
Finding patterns - they build tools to analyse these reports and look for patterns, like if a certain side effect is happening more than expected.
Reporting to authorities - they make sure all this information is sent to health authorities quickly and accurately to keep medicines safe for everyone.
Patient safety is a fundamental principle for GSK
Once drugs are produced, they need to be stored and shipped to pharmacies. Data engineering is important in ensuring this happens smoothly.

Tracking medicines - data engineers create systems to track the medicines as they are made, stored, and shipped.
Optimising inventory - they help the company figure out how much medicine to keep in stock to avoid running out or wasting any.
Monitoring shipments - they set up real-time tracking so the company knows exactly where shipments are and can fix any delivery problems quickly.'
